现象：双击 Cherry Studio 后没有窗口，进程很快退出。

崩溃转储里的关键提示：

```text
GPU process isn't usable. Goodbye.
```

临时解决方式：

```powershell
"D:\APPs\Cherry Studio\Cherry Studio.exe" --disable-gpu --disable-gpu-compositing --disable-gpu-sandbox
```

后续处理：创建一个带兼容参数的桌面快捷方式。生产类软件遇到 Electron 闪退时，可以优先检查：

- GPU 进程
- 硬件加速
- 应用用户数据目录
- Crashpad 报告
